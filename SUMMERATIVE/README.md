# SUMMERATIVE LABS FROM FLATIRON 

## Week 1: Software Engineering Fundementals
    1. Simple Java Calculator
        **Lab Objectives:  create a JavaScript program that simulates a calculator capable of performing basic operations and keep track of all calculations made.**
            1. Implement functions for addition, subtraction, multiplication, and division.
            2. Store each calculation’s details in an array.
            3. Display the history of calculations to the user.

    2. Personal Website
        **Lab Objectives: create an HTML page that will serve as a personal website.**
            1. Build a multi-section website using HTML.
            2. Style the website using CSS to improve aesthetics.
            3. Include information about you, your skills, and your projects.

## Week 2: Frontend Development Javascript
    1. Simple Quiz App 
        ** Lab Objective: implement in the terminal a test based on the user's input.**
            1.  There should be three different tests the user can select: easy, medium, or hard. 
            2. The differences between the quizes are not only based on the ease of test questions--but also the amount of time the user must answer the questions. 
            3. I provide basic error handling to control if the user does not enter a proper name, does not select a proper test, and if they desire to quit.
            4. Implement a place to provide not only feedback, but a history of the user's answers. 
            5. At the end of the quiz the user should be given a score, and presented with an option to retake the test or select a new one.

## Week 3: Frontend Development JavaScript Interactions 
    1. Single Page Application (SPA) 
        **Lab Objective: using HTML, CSS, and JavaScript design and develop a Single Page Application (SPA) for a dictionary.**
            1.  Enable users to search for a word and retrieve information (such as pronunciation, definitions, synonyms, and source details interactively.
            2. Search for Words from users input into an HTML form and fetch its definition from an external dictionary API.
            3. Display the definitions dynamically to show the word’s definition, part of speech, and example usage on the same page without a reload.
            4. The JavaScript should dynamically update the CSS (highlighting saved words or changing themes for better readability).
            5. Perform Error Handling by displaying an error message if a word is not found or if the API request fails.
